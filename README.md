# 스도쿠 솔버 (Sudoku Solver)

C++ Win32 네이티브 GUI 스도쿠 해결 프로그램입니다. 백트래킹 없이 논리적 전략만으로 퍼즐을 해결합니다.

## 다운로드

[Releases](https://github.com/needitem/Sudoku-Solver/releases)에서 `sudoku.exe` 다운로드

## 해결 전략

### 기본 전략

#### Naked Singles (단일 후보)
셀에 가능한 숫자가 하나만 남으면 확정합니다.

#### Hidden Singles (숨은 단일)
특정 숫자가 행/열/박스 내에서 한 곳에만 들어갈 수 있으면 확정합니다.

### 중급 전략

#### Naked Pairs (단일 쌍)
같은 행/열에서 동일한 2개 후보를 가진 셀 2개를 찾으면, 해당 행/열의 다른 셀에서 그 후보들을 제거합니다.

```
행: [12] [345] [12] [3456] ...
     ↑          ↑
   Naked Pair → 다른 셀에서 1,2 제거
```

#### Pointing Pairs (Box-Line Reduction)
박스 내에서 특정 숫자가 한 행/열에만 있으면, 그 행/열의 박스 외부에서 해당 숫자를 제거합니다.

```
┌─────────┐
│ . 5 . │ . . . │ . . .
│ . 5 . │       │
│ . . . │       │
└─────────┘
박스 내 5가 첫 열에만 → 첫 열의 다른 박스에서 5 제거
```

### 고급 전략

#### X-Wing
2개 행에서 특정 숫자가 같은 2개 열에만 있으면, 해당 열의 다른 행에서 그 숫자를 제거합니다.

```
     c1    c2
r1:  5     5    ← 숫자 5가 c1, c2에만 존재
r2:  5     5    ← 숫자 5가 c1, c2에만 존재
          ↓
c1, c2의 다른 행에서 5 제거
```

#### Swordfish
X-Wing의 확장. 3개 행에서 특정 숫자가 같은 3개 열에만 있으면, 해당 열의 다른 행에서 제거합니다.

#### Y-Wing (XY-Wing)
피벗 셀(AB)과 두 날개 셀(AC, BC)로 구성. 두 날개가 모두 볼 수 있는 셀에서 C를 제거합니다.

```
피벗(AB) ─── 날개1(AC)
    │
날개2(BC)

→ 날개1과 날개2가 모두 보는 셀에서 C 제거
```

## 전략 적용 순서

```
Naked Singles → Hidden Singles → Naked Pairs → Pointing Pairs → X-Wing → Swordfish → Y-Wing
      ↑_______________________________________________________________________________|
                                    (반복)
```

## 사용법

1. 숫자 입력 (1-9)
2. "해결" 클릭
3. 파란색 = 솔버가 채운 숫자

## 빌드

```cmd
cl /EHsc /O2 /utf-8 sudoku.cpp user32.lib gdi32.lib comctl32.lib /Fe:sudoku.exe
```
